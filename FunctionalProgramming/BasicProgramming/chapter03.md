# Chapter3 変数の定義

## 3.1 変数の必要性

同じ情報が複数の場所に書かれている時に，
変数を使うことで一つの変更で済ませることができる．


## 3.2 変数定義の構文

OCamlでは，以下の構文で変数を定義する．
```
let 変数名 = 式
```

変数名の最初は小文字でなければならない．


## 3.3 変数の実行方法

演算子の順にしたがって，順に変数を代入して計算を行う．

ex)

```
let a = 10 ;;

(2 * a) + (4 * a)
= (2 * 10) + (4 * a)
= 20 + (4 * a)
= 20 + (4 * 10)
= 20 + 40
= 60

```

## 3.4 ほかの言語の変数との違い

関数型言語の変数は，基本的に一度定義したら値を書き換えることはできない．
変数が変わらないことで，プログラムの見通しがよくなり，間違いが減ることでぷごらうむの信頼性が向上する．
